# Bulls and Cows

This project is a console implementation of the classic logical game **“Bulls and Cows”**, developed in **C++**.

The program allows a user to play the game against the computer in two different modes using a text-based interface.

## Features

- Classic Bulls and Cows gameplay
- Two game modes:
  - **First Mode** – the player guesses a number generated by the computer
  - **Second Mode** – the computer guesses the number chosen by the player
- Random number generation with unique digits
- Calculation of bulls (correct digit in correct position) and cows (correct digit in wrong position)
- Console interface built using pseudo-graphics
- Menu navigation using keyboard arrows and Enter
- Input validation and error handling

## Game Rules

- The hidden number consists of **four different digits (0–9)**.
- After each attempt:
  - A **bull** means a correct digit in the correct position.
  - A **cow** means a correct digit in the wrong position.
- In the first mode: the player wins if the number is guessed within the allowed number of attempts.
- In the second mode: the computer wins if the number is guessed within the allowed number of attempts.
